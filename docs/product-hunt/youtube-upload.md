# YouTube upload package

Product Hunt accepts a full YouTube URL for the optional launch video.

## Recommended upload

- Video:
  `docs/product-hunt/output/hammadev-product-hunt-launch.mp4`
- Captions:
  `docs/product-hunt/output/hammadev-product-hunt-launch.srt`
- Thumbnail:
  `docs/product-hunt/assets/youtube-thumbnail.png`
- Measured duration: approximately 1 minute 46 seconds
- Visibility: **Public** or **Unlisted**, never Private

This is the dedicated Product Hunt launch cut with natural narration and
speech-timed burned-in captions. It focuses on the continuity problem, product
workflow, local privacy model, verification proof, and launch call to action.
Do not use the separate Build Week submission video for Product Hunt.

## Title

```text
HammaDev — Project Memory for AI Coding Agents
```

## Description

```text
HammaDev is free, local project memory for Codex, Claude Code, and Grok.

Run one command, choose an agent, and continue with decisions, verification,
Git state, risks, and one clear next action intact—without a HammaDev account,
cloud backend, telemetry, or transcript upload.

Try the beta: https://hammadev.nematov.com/
Source: https://github.com/hamma-labs/hammadev
Install: npm install -g hammadev@beta
```

After upload, copy the full `https://www.youtube.com/watch?v=...` URL into
`videoUrl` in `submission.json`. Upload `captions.srt` as English captions and
check the first and final 15 seconds before adding the video to Product Hunt.
